# BnS Buddy Updater
Updater for BnS Buddy

> Made by Kogaruh aka (x)Endless

**Requirements:**
- Internet Connection. 
- MetroFramework.dll (included)

**What Does it do?:**
- Can updater BnS Buddy to latest version.
- Can download BnS Buddy's latest version.
